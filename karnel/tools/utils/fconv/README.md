# Conversor de Arquivos (fconv)

Converta entre formatos de arquivo pelo terminal

**Pacote:** fconv  
**Autor:** israel marques  
**Repositório:** https://github.com/israelmarques1024-dotcom/karnel-termux  
**Oficial:** https://github.com/dedsec1121fk/DedSec  
**Tipo:** Utilitário (Python)  
**Licença:** MIT

## Descrição

File Converter é uma ferramenta Python que converte arquivos entre vários formatos diretamente do terminal. Suporta múltiplos formatos de entrada e saída para transformações rápidas sem sair da linha de comando.

## Dependências

- Python 3
- Instalado via curl do repositório DedSec

## Instalar

```bash
karnel install utils --fconv
```

## Desinstalar

```bash
karnel uninstall utils --fconv
```

## Atualizar

```bash
karnel update utils --fconv
```
