## Descrição

Smart Notes é uma ferramenta de anotações no terminal que permite criar, organizar e pesquisar notas sem sair da linha de comando. Oferece captura rápida, categorização e busca.

## Dependências

- Python 3
- Instalado via curl do repositório DedSec

## Instalar

```bash
karnel install utils --notes
```

## Desinstalar

```bash
karnel uninstall utils --notes
```

## Atualizar

```bash
karnel update utils --notes
```
