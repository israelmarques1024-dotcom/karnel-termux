# Password Master (passman)

Gerenciador de senhas no terminal

**Pacote:** passman  
**Autor:** israel marques  
**Repositório:** https://github.com/israelmarques1024-dotcom/karnel-termux  
**Oficial:** https://github.com/dedsec1121fk/DedSec  
**Tipo:** Utilitário (Python)  
**Licença:** MIT

## Descrição

Password Master é um gerenciador de senhas no terminal para armazenar, gerar e recuperar credenciais com segurança. Oferece armazenamento criptografado e geração de senhas.

## Dependências

- Python 3
- Instalado via curl do repositório DedSec

## Instalar

```bash
karnel install utils --passman
```

## Desinstalar

```bash
karnel uninstall utils --passman
```

## Atualizar

```bash
karnel update utils --passman
```

## Notas

- Comando: `passman`
- Armazene sua senha mestre com segurança
