# Criador de Sites (websites)

Gere sites simples pelo terminal

**Pacote:** websites  
**Autor:** israel marques  
**Repositório:** https://github.com/israelmarques1024-dotcom/karnel-termux  
**Oficial:** https://github.com/dedsec1121fk/DedSec  
**Tipo:** Utilitário (Python)  
**Licença:** MIT

## Descrição

Simple Websites Creator gera estruturas básicas de sites a partir da linha de comando. Crie rapidamente templates de projetos HTML/CSS/JS sem precisar de um framework completo.

## Dependências

- Python 3
- Instalado via curl do repositório DedSec

## Instalar

```bash
karnel install utils --websites
```

## Desinstalar

```bash
karnel uninstall utils --websites
```

## Atualizar

```bash
karnel update utils --websites
```

## Notas

- Comando: `websites`
