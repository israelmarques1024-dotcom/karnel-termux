# httptmux

Cliente de API HTTP interativo para o terminal

**Pacote:** httptmux  
**Autor:** israel marques

**Oficial:** https://www.npmjs.com/package/httptmux

**Licença:** MIT

## Descrição

httptmux é um cliente de API HTTP interativo que roda no terminal. Construa e envie requisições HTTP, inspecione respostas e gerencie fluxos de trabalho de API sem sair da linha de comando. Similar ao Postman, mas para o terminal.

## Dependências

- Node.js (npm)
- Instalado via npm globalmente

## Instalar

```bash
karnel install utils --httptmux
```

## Desinstalar

```bash
karnel uninstall utils --httptmux
```

## Atualizar

```bash
karnel update utils --httptmux
```

## Notas

- Comando: `httptmux`
- Requer Node.js (instalado automaticamente se ausente)
