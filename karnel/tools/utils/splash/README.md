# Tela de Carregamento (splash)

**Autor:** israel marques

## Descrição

Loading Screen fornece telas de carregamento e animações personalizáveis para scripts e aplicações no terminal. Adicione feedback visual a tarefas longas com indicadores de progresso e banners.

## Dependências

- Python 3
- Instalado via curl do repositório DedSec

## Instalar

```bash
karnel install utils --splash
```

## Desinstalar

```bash
karnel uninstall utils --splash
```

## Atualizar

```bash
karnel update utils --splash
```
