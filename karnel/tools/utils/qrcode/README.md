# Gerador de QR Code (qrcode)

Gere QR codes a partir de texto e URLs

**Pacote:** qrcode  
**Autor:** israel marques  
**Repositório:** https://github.com/israelmarques1024-dotcom/karnel-termux  
**Oficial:** https://github.com/dedsec1121fk/DedSec  
**Tipo:** Utilitário (Python)  
**Licença:** MIT

## Descrição

QR Code Generator cria QR codes a partir de texto, URLs ou conteúdo da área de transferência diretamente no terminal. Gere rapidamente QR codes compartilháveis para links, credenciais Wi-Fi, informações de contato e muito mais.

## Dependências

- Python 3
- Instalado via curl do repositório DedSec

## Instalar

```bash
karnel install utils --qrcode
```

## Desinstalar

```bash
karnel uninstall utils --qrcode
```

## Atualizar

```bash
karnel update utils --qrcode
```
