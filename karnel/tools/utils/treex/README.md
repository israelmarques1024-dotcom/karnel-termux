# Explorador de Árvore (treex)

Visualização interativa de árvore de diretórios

**Pacote:** treex  
**Autor:** israel marques  
**Repositório:** https://github.com/israelmarques1024-dotcom/karnel-termux  
**Oficial:** https://github.com/dedsec1121fk/DedSec  
**Tipo:** Utilitário (Python)  
**Licença:** MIT

## Descrição

Tree Explorer fornece visualização interativa de árvore de diretórios com filtragem, pesquisa e exportação. Navegue e entenda a estrutura do seu projeto mais facilmente do que com o `tree` simples.

## Dependências

- Python 3
- Instalado via curl do repositório DedSec

## Instalar

```bash
karnel install utils --treex
```

## Desinstalar

```bash
karnel uninstall utils --treex
```

## Atualizar

```bash
karnel update utils --treex
```
