# DedSec Network Toolkit

Kit de ferramentas multifuncional de rede, OSINT e pentest

**Pacote:** dedsec-network  
**Autor:** israel marques  
**Repositório:** https://github.com/israelmarques1024-dotcom/karnel-termux  
**Oficial:** https://github.com/dedsec1121fk/DedSec  
**Tipo:** Ferramenta de rede (Python)  
**Licença:** MIT

## Descrição

DedSec Network Toolkit combina varredura de rede, coleta OSINT e utilitários de teste de penetração em uma única ferramenta Python. Projetado para profissionais de segurança e hackers éticos.

## Dependências

- Python 3
- Instalado via curl do repositório DedSec

## Instalar

```bash
karnel install network --dedsec-network
```

## Desinstalar

```bash
karnel uninstall network --dedsec-network
```

## Atualizar

```bash
karnel update network --dedsec-network
```

## Notas

- Comando: `dedsec-network`
- Use apenas em sistemas que você possui ou tem permissão explícita para testar
