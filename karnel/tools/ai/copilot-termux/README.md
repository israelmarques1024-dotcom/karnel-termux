# Copilot-Termux

GitHub Copilot CLI for Termux (Android aarch64).

## Installation

```bash
npm i -g @bash0816/copilot-termux
```

Or via Karnel:

```bash
karnel install ai --copilot-termux
```

## Usage

```bash
copilot -p "how do I list files in Node.js?"
copilot -p "explain this shell script"
```

First-time setup:

```bash
copilot-termux setup
```

## Integration

- Registered in `karnel list ai`
- Scanned by `karnel ia sessions`
- Listed in `karnel ia routes`
- Checked by `karnel doctor`
