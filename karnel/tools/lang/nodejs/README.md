# Node.js LTS

JavaScript runtime environment (Long Term Support version)

**Package:** nodejs-lts  
**Author:** israel676767  
**Repository:** https://github.com/israel676767/karnel-termux  
**Official:** https://nodejs.org  
**Type:** Language runtime (pkg)  
**License:** MIT

## Description

Node.js is an open-source, cross-platform JavaScript runtime environment that executes JavaScript code outside a web browser. The LTS (Long Term Support) version provides a stable and reliable platform for building server-side applications, command-line tools, and more.

## Dependencies

- Installed via pkg

## Install

```bash
karnel install lang --nodejs
```

## Uninstall

```bash
karnel uninstall lang --nodejs
```

## Update

```bash
karnel update lang --nodejs
```
