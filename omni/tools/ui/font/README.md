# Meslo Nerd Font

Patched font with Nerd Font icons for Termux

**Package:** omni-catalyst-assets (font)  
**Author:** israel676767  
**Repository:** https://github.com/israel676767/omni  
**Official:** https://github.com/ryanoasis/nerd-fonts  
**Type:** Termux UI customization  
**License:** MIT

## Description

Meslo Nerd Font is a patched version of Meslo LG that includes Nerd Font icons and glyphs. It provides thousands of additional icons for file types, git status, powerline symbols, and more, enhancing the visual experience in Termux.

## Dependencies

- Termux (base installation)

## Install

```bash
omni install ui --font
```

## Uninstall

```bash
omni uninstall ui --font
```

## Update

```bash
omni update ui --font
```

## Notes

- Installed to `~/.termux/font.ttf`
- Restart Termux after installation to apply
- Bundled with Omni Catalyst assets
