### Kimchi CLI

Terminal coding agent powered by Kimchi's multi-model orchestration by CAST AI.

**Repository:** https://github.com/getkimchi/kimchi  
**Docs:** https://docs.kimchi.dev  

**Install:** `omni install ai --kimchi-code`  
**Run:** `kimchi`  
**Setup:** `kimchi setup` (interactive first-time configuration wizard)  
**Switch model:** `/model` or `Ctrl+P` inside the interactive CLI  

Kimchi is a standalone terminal coding agent — not an OpenCode plugin.
It configures AI coding assistants to use CAST AI's open-source models.
