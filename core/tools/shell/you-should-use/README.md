# Zsh-you-should-use

Suggests alternative commands when you use outdated ones

**Package:** zsh-you-should-use  
**Author:** DevCoreX  
**Repository:** https://github.com/DevCoreXOfficial/omni-catalyst  
**Official:** https://github.com/MichaelAquilina/zsh-you-should-use  
**Type:** ZSH plugin (git clone)  
**License:** GPL-3.0

## Description

Zsh-you-should-use reminds you when there are alternative, more modern commands available for what you typed. For example, it suggests `bat` if you type `cat`, or `lsd` if you type `ls`.

## Dependencies

- ZSH, git, zoxide

## Install

```bash
core install shell --you-should-use
```

## Uninstall

```bash
core uninstall shell --you-should-use
```

## Update

```bash
core update shell --you-should-use
```

## Notes

- Installed in `~/.zsh-plugins/`
- Suggests modern alternatives
- Customizable triggers and messages
